Try Demo

About the Project

This project implements a deadlock detection system using Resource Allocation Graphs (RAG). It provides a visual interface for understanding and simulating deadlock scenarios in operating systems.

The system uses Depth-First Search (DFS) to detect cycles in the graph, which is the standard method for deadlock detection in operating systems.

Key Features

Interactive RAG

Visual representation of Resource Allocation Graphs with interactive nodes and edges.

Real-time Detection

Instant deadlock detection using DFS algorithm with visual feedback.

User-Friendly

Intuitive interface with easy node creation and edge management.

Open Source

Fully open-source implementation with clean, documented code.

How to Run the Demo

  1. Clone the repository: git clone https://github.com/Saatvik-Cheruku/automated-deadlock-detector.git
  2. Install dependencies: pip install -r requirements.txt
  3. Run the application: cd gui
    python main.py

Controls:

  • Press '1' and click to create process nodes (green)
  • Press '2' and click to create resource nodes (red)
  • Left-click nodes to create edges
  • Right-click nodes to delete them
  • Click to detect cycles
